# 快速开始
OpenComputers是一个在游戏中加入计算机和机器人的 mod。

它可以用 [Lua](http://www.lua.org/) [5.3](http://www.lua.org/manual/5.3/manual.html) 进行编程。

它采用了其他几个 mod 的创意，例如 ComputerCraft, StevesCarts 和 Modular Powersuits，并创造出了一些好玩的新东西。
